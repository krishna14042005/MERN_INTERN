how to use "setOnInsert in mongoDB"?

The $setOnInsert operator in MongoDB is used in conjunction with the update or updateOne/updateMany methods with the upsert option set to true. It allows you to specify fields that should be set only when a new document is inserted. If the document already exists, $setOnInsert does nothing.

works with upsert: true: If no document matches the filter, MongoDB inserts a new document and applies the $setOnInsert fields. If a document matches, $setOnInsert is ignored.
Does not overwrite existing fields: Fields specified in $setOnInsert are only applied during insertion, not during updates.
